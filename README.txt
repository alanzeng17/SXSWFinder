Instructions for running SXSWFinder

This needs to be run on Python 2.X

1. Open terminal, and navigate to where main.py is stored.
2. type: python main.py <Spotify Username>
2b. (OPTIONAL) add a “> out.txt” without the quotations to the above command in order to pipe the output to a text file.
3a. Authorize access if prompted, and copy the link it redirects you to. It should look like http://localhost…
3b. Paste that link into the command line. Once done, you should not have to do this again.
4.Follow the prompts given in the terminal to get desired output.
